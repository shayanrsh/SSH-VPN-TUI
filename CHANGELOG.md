# Changelog

All notable changes to this project will be documented in this file.

The format is based on Keep a Changelog, and this project adheres to Semantic Versioning.

## [1.0.0] - 2026-05-27
### Added
- Textual-based admin TUI with dashboard, user form, and user detail screens
- SQLite persistence for VPN user metadata, traffic counters, and audit events
- Linux user management with forced no-login shell and sshd Match blocks
- Traffic accounting using iptables with limit enforcement and resets
- Background scheduler for traffic sampling and expiry enforcement
- Installer, updater, and uninstaller scripts
- GitHub issue templates and MIT license
